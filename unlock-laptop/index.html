Cẩm Nang Windows & Laptop

Tra cứu nhanh các thủ thuật, phím tắt và cách khắc phục lỗi trên Windows từ sổ tay gốc.

1. Sửa Lỗi Nhanh Windows
    [cite_start]
  • Khởi động lại máy: Giúp giải quyết nhiều lỗi tạm thời. [cite: 2, 3]
    • [cite_start]
  • Kiểm tra cập nhật: Vào Settings → Windows Update → Check for updates để vá lỗi bảo mật. [cite: 4, 5, 6]
    • [cite_start]
  • Quét file hệ thống hỏng: Mở CMD (Admin) và nhập `sfc /scannow` hoặc `DISM /Online /Cleanup-Image /RestoreHealth`. [cite: 11, 12, 14, 16]
    • [cite_start]
  • Kiểm tra RAM & Ổ cứng: Dùng lệnh `chkdsk C: /f` cho ổ cứng và "Windows Memory Diagnostic" cho RAM. [cite: 22, 23, 24]
    • [cite_start]
  • Màn hình xanh (BSOD): Ghi lại mã lỗi để tra Google hoặc dùng công cụ BlueScreenView phân tích. [cite: 43, 44]
2. Mở, Đóng & Bảo Quản Laptop
    [cite_start]
  • Mở/đóng máy: Dùng 2 tay từ chính giữa, không mở bằng 1 tay ở góc để tránh hỏng bản lề. [cite: 55, 57, 58, 94]
  • Chế độ Sleep: Dùng khi tạm ngưng 15-30 phút (nhắm màn hình). [cite_start]Tiêu thụ ít điện, khôi phục nhanh. [cite: 84, 87, 113, 114]
  • Chế độ Hibernate: Dùng khi tạm ngưng 2-8 giờ. [cite_start]Lưu trạng thái vào ổ cứng, không tốn điện. [cite: 88, 90, 92, 115, 116]
    • [cite_start]
  • Lưu ý quan trọng: Không rút pin đột ngột khi đang bật, không che kín lỗ thông hơi. [cite: 107, 108, 109]
3. Phím Tắt Đa Nhiệm & Hệ Thống
    [cite_start]
  • Chuyển ứng dụng: `Alt + Tab`. [cite: 183]
  • Task View & Desktop ảo: `Win + Tab` để mở Task View. [cite_start]Dùng `Win + Ctrl + D` để tạo Desktop mới. [cite: 182, 192, 193]
    • [cite_start]
  • Chia đôi màn hình: `Win + ⬅️` hoặc `Win + ➡️` (Snap Windows). [cite: 187, 188, 189]
    • [cite_start]
  • Lịch sử Clipboard: `Win + V` để mở lịch sử sao chép. [cite: 272, 273]
    • [cite_start]
  • Chụp màn hình: `Win + Shift + S` (Chọn vùng tùy ý) hoặc `Win + PrtSc` (Chụp toàn màn hình tự lưu). [cite: 279, 339, 340]
    • [cite_start]
  • Emoji: `Win + .` (Dấu chấm) để mở bảng emoji. [cite: 296, 297]
4. Lệnh Command Line (CMD) Hữu Ích
    [cite_start]
  • Kiểm tra mạng: `ipconfig` (Xem IP), `ping google.com` (Kiểm tra kết nối internet). [cite: 209, 210, 211, 212, 213]
    • [cite_start]
  • Quản lý tiến trình: `tasklist` (Xem danh sách) và `taskkill /F /IM notepad.exe` (Tắt ứng dụng ép buộc). [cite: 224, 225, 226, 227]
    • [cite_start]
  • Sửa lỗi mạng: Khi mạng bị lỗi, đặt lại TCP/IP bằng `netsh winsock reset` và `netsh int ip reset`. [cite: 46, 47, 48]
5. Tối Ưu Hóa Hiệu Suất & Bảo Mật
    [cite_start]
  • Startup Optimization: Mở Task Manager (`Ctrl + Shift + Esc`) → Thẻ Startup → Vô hiệu hóa ứng dụng không cần thiết. [cite: 437, 438, 439]
    • [cite_start]
  • Dọn rác (Disk Cleanup): Tìm "Disk Cleanup" trong Start, chọn "Clean up system files" để xóa file hệ thống thừa. [cite: 441, 442, 448]
    • [cite_start]
  • Chống phân mảnh: Tìm "Defragment and Optimize Drives" (Chỉ dành cho HDD, SSD không cần làm). [cite: 449, 450, 453]
  • Windows Defender: Luôn bật Real-time protection và Firewall. [cite_start]Chạy Quick scan để quét virus nhanh. [cite: 399, 401, 402, 403]
